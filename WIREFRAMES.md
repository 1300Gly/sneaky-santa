## Responsive Breakpoints

### Desktop (lg: 1024px+)
- 3-column layout for game screen
- Dice Rules: 2 columns
- Full card width for card picker

### Tablet (md: 768px - 1023px)
- 2-column layout for game screen
- Dice Rules: 2 columns
- Stacked timer and dice

### Mobile (< 768px)
- Single column layout
- Stacked components
- Full-width cards
- Dice Rules: 1 column

---

## Interaction States

### Button States
- **Default**: Background color with border
- **Hover**: Darker background, elevated shadow
- **Active**: Scale down (0.95)
- **Disabled**: Reduced opacity (0.5)

### Card States
- **Unrevealed**: Red gradient, hover scale (1.05)
- **Revealed**: White background, gold border, flip animation
- **Hover**: Elevated shadow

### Modal States
- **Opening**: Scale from 0.95 to 1.0, fade in
- **Closing**: Scale down, fade out
- **Backdrop**: Dark overlay (50% opacity)

---

## Animation Notes

1. **Card Flip**: 3D rotation (0.6s cubic-bezier)

3. **Modal**: Scale + fade (0.3s)
4. **Button Press**: Scale down (0.95)
